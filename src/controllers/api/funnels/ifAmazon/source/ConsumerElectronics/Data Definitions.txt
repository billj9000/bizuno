    parent_child    Parentage    Defines the product within a relationship with another, similar product. The "parent" defines the basic data about the product. The "child" products define the possible variations (i.e., size, color) of the parent product and represent the items that are actually for sale. This field is required if any of the following columns are populated: variation_theme, color, paper_size, maximum_expandable_size, line_size.
