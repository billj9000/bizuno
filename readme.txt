== Description ==

Bizuno Accounting, by PhreeSoft, is a full featured CRM/ERP application. More information is available at the <a href="https://bizuno.com">Bizuno</a> website:

== Installation ==

1. This version of the plugin is only a placeholder and should allow for the download of the latest library directly from the Bizuno.com website. 
1. Once updated, activate the plugin through the 'Plugins' screen in the WordPress admin page
1. Click on the Bizuno menu item to load the Bizuno Portal.
1. From the Welcome portal, click the Install button to configure the application and install the database tables. A pop up will appear to enter some initial information, select your preferences and press the Next icon.
1. Wait about 10 seconds to get everything up and configured and the page will reload with your new Bizuno home page dashboard.
1. Upgrades are handled via the standard WordPress upgrade utility.

= Minimum Requirements =

* PHP   version 8.0 or greater (PHP 8.2 or greater is recommended, tested with PHP 8.4)
* MySQL version 5.0 or greater (MySQL/MariaDB X.X or greater is recommended, tested with MariaDB X.X)
* WordPress 6.5+
